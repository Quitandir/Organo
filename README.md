Organo is an application designed to manage people and teams from the same organization, assigning each one of them to different teams according to your management needs.  

## Access

https://quitandir.github.io/Organo/

## About the project

Organo is an application developed during the first course from the React formation at Alura Escola de Tecnologia. From the Figma model received, the course started the basics over creating a react application, explained the concept of React components and how to build the app using its main features. 

For this project, JavaScript was the language chosen, but this sequence of courses on React has a moment in which TypeScript will be introduced and then, future projects with React will be done using TypeScript.
